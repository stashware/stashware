# Building from source  

## Requirements  

- Golang 1.11+  
- SQLite3 headers  

``` console
$ git clone https://github.com/stashware/stashware.git  
$ cd stashware  
```

You will then find the gzipped tarball at stashware-x.y.x.tgz
